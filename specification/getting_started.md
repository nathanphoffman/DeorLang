# Getting Started
To get started with Deor you will need to install cargo/rust, then pull the latest version from the git repo.

## Install Cargo (if you don't have it)
It is worth [visiting this link](https://rust-lang.org/tools/install/) for the latest cargo instructions, but below is how the cargo install worked on Linux as of Q3 2026 for Linux

### Linux Cargo Install
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Pull Latest Deor From GitHub
```
git
```
